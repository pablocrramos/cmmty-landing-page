¿Qué padding horizontal debe tener el contenido dentro de las líneas?

El espacio entre las líneas guía (1px) y el contenido real de cada sección.

MOBILE <768px / TABLET 768-1023px / DESKTOP 1024px+

Tablet

Desktop

A — Escalonado suave (px-4 / px-6 / px-8)

16px mobile · 24px tablet · 32px desktop. Igual al patrón mayoritario actual. Incremento gradual.

B — Más respiro (px-6 / px-8 / px-12)
